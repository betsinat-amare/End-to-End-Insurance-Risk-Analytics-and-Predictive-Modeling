# Insurance Risk Analytics

## Project Overview
Analyzing historical car insurance data to identify low-risk segments and optimize premiums.

## Data Description
- Policy data
- Client data
- Car data
- Location data
- Payment & claims data

## Tasks
1. Exploratory Data Analysis (EDA)
2. Statistical Analysis & A/B Testing
3. Machine Learning Modeling
4. Data Version Control (DVC)
